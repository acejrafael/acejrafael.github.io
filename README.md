# Jacques Wainright Portfolio

Static GitHub Pages portfolio for `acejrafael.github.io`.

The `main` branch serves one final single-page portfolio from `/`.

- Dark mode is the default, with a light mode toggle.
- English, French, and Central America Spanish interface copy are included.
- LinkedIn is featured as the primary social profile.
- The public resume download uses the master resume only, available in English, French, and Spanish.
- Three recommended template packs are included in English, French, and Spanish: project management, marketing CRM/campaigns, and AI automation/workflow systems.
- Skills and selected certifications are listed directly on the page.

No CRM evidence, Gmail evidence, source maps, local Desktop paths, phone numbers, DOCX resumes, raw proposals, unpublished article pages, or private IDs are included.
